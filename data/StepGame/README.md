# StepGame

StepGame was created to test PLM abilities on multi-hop spatial reasoning.

The data is synthetic and can be generated at this [repo](https://github.com/ZhengxiangShi/StepGame). We didn't generate new data but used the one made available by the authors [here](https://huggingface.co/datasets/ZhengyanShi/StepGame).

## Adapting the original dataset Dataset

Just run the notebook **codes/join.ipynb**, to have the Hugging Faces dataset.
